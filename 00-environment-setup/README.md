# Environment setup

Get your machine ready for LLM engineering: Python, API keys, and optional local tools.

## What you need

- Python 3.10+ (recommended 3.11 or 3.12)
- A code editor (VS Code, Cursor, etc.)
- API access to at least one LLM provider (e.g. OpenAI, Anthropic) for labs

## Quick start

1. Create a virtual environment: `python -m venv .venv`
2. Activate it: `.venv\Scripts\activate` (Windows) or `source .venv/bin/activate` (Mac/Linux)
3. Install dependencies per lab: each `labs/` folder may have its own `requirements.txt`

## Optional

- Git for version control
- Docker if you plan to run open-source models or RAG stacks locally

Once set up, start with the [LLM Engineering](./01-llm-engineering/README.md) track.

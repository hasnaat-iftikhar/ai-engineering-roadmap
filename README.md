# LLM Engineering Roadmap

A structured learning roadmap, notes, and resources for LLM Engineering (frontier models, RAG, fine-tuning, agents, and production AI systems).

## Who this is for

- Engineers who want a clear learning sequence
- Builders who want practical, system-level understanding
- Beginners who want structured notes with curated links

## How to use this repo

1. Start with [Environment setup](./00-environment-setup/README.md) if needed.
2. Pick a track (LLM Engineering, RAG, Fine-tuning, Agents).
3. Read `notes.md` first, then `resources.md` for each topic.
4. Try the `labs/` demos when available.
